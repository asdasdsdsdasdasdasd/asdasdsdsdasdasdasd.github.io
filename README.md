# Wan Chi Wai — Personal Site

Persona 3–styled portfolio built with [persona3-website](https://github.com/blairxu13/persona3-website) (React + Vite).

## Local dev

```bash
npm install
npm run dev
```

Open the URL shown in the terminal. Use **↑/↓ + Enter** to navigate menus.

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` deploy automatically to GitHub Pages via Actions.

Live site: **https://asdasdsdsdasdasdasd.github.io**

## Customize

| File | Content |
|---|---|
| `src/AboutMe.jsx` | Bio panels |
| `src/ResumePage.jsx` | Education / resume list |
| `src/Socials.jsx` | GitHub, email, social links |
| `src/App.jsx` | Routes + project links |
| `src/P3Menu.jsx` | Main menu labels |

Assets live in `src/assets/` (from the template author's Google Drive bundle).

**Background music:** `public/music/menu-bgm.webm` — replace to change track. Toggle with **MUSIC** or **M** key.
